# Contributing to ClearSight-ScanSense

As this project focuses on human health, our contribution process prioritizes **reproducibility** and **scientific rigor**.

## 🧬 Research Standards
* **Data Citations:** Any dataset proposed must include its origin, license (e.g., CC BY 4.0), and ethical approval status.
* **Literature Summary:** When suggesting a model, please provide a link to the peer-reviewed paper or technical preprint.
* **Technical Discussions:** Use GitHub Issues to document architectural choices (e.g., why choosing a specific loss function like Dice Loss over Cross-Entropy).

## 💻 Code Standards
* Maintain modularity: Decouple data loading (Dataloaders) from model logic.
* Type hinting is encouraged for all Python functions to ensure codebase stability.
* Document all preprocessing steps, as they are as critical as the model itself in medical AI.

## 🚀 Getting Started
1. Check the **Projects** tab for the "Literature Review" task.
2. Share your initial findings on medical imaging benchmarks in the comments.
